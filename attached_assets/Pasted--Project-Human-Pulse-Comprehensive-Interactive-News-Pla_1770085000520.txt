# Project: Human Pulse (Comprehensive Interactive News Platform)

## 1. Project Overview & Vibe
Build a **React + Vite** application with **Three.js (R3F)**.

* **Atmosphere:** Relaxing, Fluid, Glassmorphism, Dreamy Light.
* **Core Tech:** React, Tailwind CSS, React Three Fiber, Zustand, Framer Motion, Recharts (for Admin).

## 4. Technical Constraints & Optimization
1.  **Mobile Optimization:**
    * Stack the 5 spheres vertically or in a tighter cluster on mobile screens (`max-width: 768px`).
    * Ensure Glassmorphism UI is readable on small screens.
2.  **Performance:**
    * Use `InstancedMesh` for particles.
    * Use `React.memo` for the Admin Dashboard charts.
3.  **File Structure:**
    * `src/components/3d/...` (Scene, Spheres, Particles)
    * `src/pages/...` (Home, MyPage, Journalist, Admin)
    * `src/components/ui/...` (Chatbot, Modal, Navbar)
    * `src/store/useStore.js` (Global State)
