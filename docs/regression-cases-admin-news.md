# Admin/News Regression Cases (20)

## 범위
- Admin: 숨김/게시, 이슈 CRUD, 메모, 삭제, Export
- News/Emotion: 목록 노출, 숨김 반영, 상세 모달, 저장/공유/인사이트

## 실행 조건
- 환경: `npm run dev` 실행 상태, 최신 DB 마이그레이션 반영
- 브라우저: Chrome 기준 (Desktop/Tablet/Mobile viewport)
- 사용자: 비로그인/로그인(일반)/관리자 계정 분리 검증

## 테스트 케이스
1. Admin 목록 로드
- 절차: 관리자 페이지 진입
- 기대: 기사/상태/이슈수/메모 요약 노출

2. 숨김 처리 버튼 동작
- 절차: 게시중 기사에서 숨김 버튼 클릭
- 기대: 상태 배지가 `숨김 처리됨`으로 변경

3. 재게시 버튼 동작
- 절차: 숨김 기사에서 게시하기 버튼 클릭
- 기대: 상태 배지가 `게시중`으로 변경

4. 숨김 데이터 News 반영
- 절차: 특정 기사 숨김 후 감정 목록 새로고침
- 기대: 해당 기사 목록 미노출

5. 숨김 해제 반영
- 절차: 재게시 후 감정 목록 새로고침
- 기대: 기사 재노출

6. 검수완료 토글
- 절차: 검수완료 버튼 클릭
- 기대: `검수완료` 배지 즉시 반영

7. 이슈 등록
- 절차: 이슈등록 버튼 클릭 후 텍스트 입력
- 기대: 이슈 수 +1, 최신 이슈 노출

8. 이슈 수정
- 절차: 상세 팝업에서 이슈 수정 버튼
- 기대: 수정 텍스트 저장 및 반영

9. 이슈 삭제
- 절차: 상세 팝업에서 이슈 삭제 버튼
- 기대: 해당 이슈 제거, 수량 감소

10. 메모 저장
- 절차: 메모 버튼 클릭 후 텍스트 입력
- 기대: 메모 요약 라벨 노출 및 유지

11. 메모 새로고침 유지
- 절차: 메모 저장 후 새로고침
- 기대: 메모 유지

12. 기사 삭제
- 절차: 삭제 버튼 클릭
- 기대: 목록에서 제거, 재조회 시 미존재

13. Excel Export
- 절차: Excel 버튼 클릭
- 기대: 파일 생성, 상태/검수/이슈/메모 컬럼 포함

14. PDF Export
- 절차: PDF 버튼 클릭
- 기대: 파일 생성, 주요 컬럼 포함

15. 모바일 상단 3버튼 레이아웃
- 절차: 모바일 뷰 확인
- 기대: 1행 3열 유지, 버튼 눌림 영역 확보

16. 모바일 액션 아이콘 툴팁
- 절차: 액션 아이콘 hover/focus
- 기대: 기능명 확인 가능

17. 상세 모달 열기/닫기
- 절차: 카드 클릭 -> 닫기
- 기대: 모달 정상 열림/닫힘, 스크롤 잠금 복원

18. 저장 액션
- 절차: 상세에서 저장 클릭
- 기대: 저장 완료 토스트 노출

19. 공유 액션
- 절차: 상세에서 공유 클릭
- 기대: 공유 완료/대체 처리 메시지 노출

20. 인사이트 저장
- 절차: Add Insight -> 텍스트 입력 -> 저장
- 기대: 커뮤니티 등록 성공 또는 명확한 에러 표시

## 결과 기록 템플릿
- 케이스 번호:
- 결과: `PASS` | `FAIL` | `BLOCKED`
- 실제 결과:
- 증빙: 스크린샷/로그 링크
- 원인(cause):
- 대안(workaround):
- 영향(impact):

## 실행 체크리스트
- 브레이크포인트: Desktop(>=1280), Tablet(768~1279), Mobile(<=767)
- 권한: 로그인 사용자/비로그인 사용자 각각 검증
- 실패 시 기록: cause/workaround/impact

## 완료 기준 (DoD)
- 20개 케이스 모두 실행
- Blocker(숨김 반영 실패, 복귀 동선 실패, 데이터 유실) 0건
- FAIL 케이스는 원인/우회/영향 기록 완료
