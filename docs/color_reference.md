# Hue Brief Color Reference

Source: `docs/Color_palatte.png`

## Category Base Colors
- `immersion`: `#f4606b` (Coral Red)
- `vibrance`: `#ffd150` (Yellow)
- `serenity`: `#88d84a` (Green)
- `clarity`: `#3f65ef` (Light Navy Blue)
- `gravity`: `#bababa` (Neutral Gray)

## News Card Text Rules
- `title`: always `#232221` across all depth steps
- `body_low_step`: `#5f5d5c` (improved readability on LOW background)
- `body_mid_step`: `#787674`
- `body_high_step`: `rgba(255,255,255,0.84)`
